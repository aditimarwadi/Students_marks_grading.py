# Students Marks Grading System (Python)

A simple Python CLI program that assigns grades based on student marks. The user inputs their marks, and the program calculates the grade according to predefined criteria: Ex, A, B, C, D, F. This project demonstrates Python fundamentals, conditional logic, and input handling.

## Features
- Input student marks via command line
- Assigns grades based on marks range
- Provides instant feedback for students

## Skills Demonstrated
- Python basics (variables, input, print)
- Conditional statements and logic
- CLI-based application

## How to Use
1. Run `marks_grading.py`
2. Enter marks when prompted
3. View the assigned grade
